<h1>About</h1>
<p>This repository has been created as a to send sms with Twilio api.<p>

<h2>Technologies</h2>
<li>Express</li>
<li>Twilio</li>

<h2>Requirements<h2>
<li>Twilio Membership</li>

<br>
<h3>Installation</h3>

 `$ git clone https://github.com/thebozturk/send-sms-app.git`

 `$ cd send-sms-app`
 
 `$ npm install `
 
  `$ npm start `
 <br>
 
 ---
> **_NOTE:_**
You have to create your own `env` file for api keys.
